Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

Business Logic URL HTTP Method Request body data format Response body data format
Get list of all Movies /movies GET None A JSON object holding list with data about all the Movies
Get data about Movie by title /movies/[Title] GET None A JSON object holding data about a single movie,containing a title, director,year,description,
genre,leadingMan, leadingFemale, Example:

{
title: "Out of Africa",
director: "Sydney Pollac",
year: "1985",
description: "In 20th century colonial Kenya,
a Danish plantation owner has a passionete love affair,
with a free-spirited big game hunter.",
genre: ["Drama", "Biography" "Adventure"],
leadingMan: "Robert Redford",
leadingFemale: "Meryl Streep"
},
Get data about Genre by name /genres/[Name] GET None A JSON object holding data about a movie genre

{
"name": "Adventure",
"description": "Adventure films are a genre of film that typically use their action scenes to display and explore exotic locations in an energetic way."
}
Get data about Director by name /directors/[Name] GET None A JSON object holding data about a director

{
"name": "Sydney Pollack",
"bio": "",
"birthyear": "",
"deathyear": ""
}
Add a new User /users POST A JSON object holding data about the user to add, structured like:

{
"name": "Daniel2",
"username": "ama",
"password": "ama12",
"email": "daniela@yahoo.com",
"birthday": "",
"favourites": "27",
}

 A JSON object holding data about the user that was added, including an ID, structured like:

{
"name": "Daniel2",
"username": "ama",
"password": "ama12",
"email": "daniela@yahoo.com",
"birthday": "",
"favourites": "27",
"id": "0ed561be-30a2-45cd-a9e6-319252abf7b5"
}
Delete User from list of users by ID /users/[ID] DELETE None A message indicating thar user was removed from the user's list
Get list of all Users /users GET None A JSON object holding list with data about all the Users
